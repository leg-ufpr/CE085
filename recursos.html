Recursos computacionais

O curso prevê conteúdo computacional. Nas páginas do curso, você verá um botão escrito Code acima de algumas tabelas e gráficos. Ao clicar nesse botão, é revelado o código que foi utilizado para gerar aquele resultado, em linguagem R. O uso de programas computacionais auxilia o entendimento do conteúdo do curso, e os códigos em R servem como ilustração.

0.1 Programa para estatística

  • The R project for Statistical Computing: página do programa R
    • Curso sobre o uso do R no LEG
    • R-Studio um ambiente para facilitar uso do R
    • R-br é a lista de discussão em português sobre o uso do R

0.2 Programas para matemática simbólica

  • Maxima (e interface wxmaxima)
  • Yacas

0.3 Outros recursos

  • LibreOffice suíte de programas semelhante ao MS-Office (editor de texto, banco de dados, apresentação e planilha) gratuito
  • Google sheets. Para instalar o add-on de estatística: - Add-Ons -> Get Add-Ons -> Procure por “Statistics” -> Procure o add-on de “Google Statisticians and Engineers”
